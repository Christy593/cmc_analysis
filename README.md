# CMC Analysis: Language Choice in AI Ideation

## Project Overview

This project analyzes how language choice (native language L1 vs English L2) affects users' experience when interacting with AI chatbots for academic problem-solution ideation.

The study focuses on:
- Problem Understanding
- Idea Development
- Solution Evaluation
- Efficiency
- Agency

We combine:
- Survey data (self-reported experience)
- Interaction logs (prompt behavior, outputs)

---

## Key Research Question

How does interacting with AI in L1 vs L2 impact:
1. Perceived ideation performance?
2. Behavioral interaction patterns?
3. Relationships between cognitive dimensions (e.g., understanding → efficiency)?
